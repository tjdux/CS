## From

- 유저 에이전트의 이메일 정보
- 검색 엔진 같은 곳에서 주로 사용
- 요청에서 사용
- 일반적으로 잘 사용되지 않음
  <br/>

## Referer

- 현재 요청된 페이지의 이전 웹 페이지 주소
- A -> B로 이동하는 경우 B를 요청할 때 `Referer: A`를 포함해서 요청
- 유입 경로 분석 가능
- 요청에서 사용
- ⚠️ Referrer의 오타
  <br/>

## User-Agent

- 클라이언트의 애플리케이션 정보 (웹 브라우저 정보 등등..)
- 통계 정보
- 어떤 종류의 브라우저에서 장애가 발생하는지 파악 가능
- 요청에서 사용
  <br/>

## Server

- 요청을 처리하는 Origin 서버의 소프트웨어 정보
- 응답에서 사용
  <br/>

## Date

- 메시지가 발생한 날짜와 시간
- 응답에서 사용
